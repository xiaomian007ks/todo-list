# 待办事项Web应用 - 简约温馨的任务管理工具

## 项目介绍

这是一个功能完整、设计精美的待办事项Web应用,采用简约温馨的风格设计,帮助您高效管理日常任务。

## 核心功能

### ✅ 任务管理
- 创建、编辑、删除待办事项
- 设置任务优先级(高/中/低)
- 设置截止日期和时间
- 添加任务描述

### 🔔 智能提醒
- 设置提醒时间
- 三种温馨铃声可选(温暖/清新/轻柔)
- 浏览器通知支持
- 自动铃声提醒

### 📅 智能分类
- 自动按日期分类(今天/明天/本周/本月/已过期/无日期)
- 状态筛选(全部/进行中/已完成/未完成)
- 多种排序方式(按优先级/日期/创建时间)

### 💾 数据持久化
- LocalStorage本地存储
- 数据自动保存
- 刷新页面不丢失数据

## 设计特点

### 🎨 温馨配色
- 珊瑚橙主色(#FF6B6B) - 温暖活力
- 薄荷绿辅助色(#4ECDC4) - 清新自然
- 米白背景(#FAF9F6) - 舒适柔和

### ✨ 流畅动画
- 任务卡片淡入动画
- 悬停上浮效果
- 状态切换过渡
- 删除滑动动画

### 📱 响应式设计
- 桌面端: 侧边栏展开布局
- 平板端: 侧边栏可折叠
- 移动端: 单列布局,底部导航

## 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **样式**: Tailwind CSS 3.4
- **状态管理**: Zustand
- **图标**: Lucide React
- **字体**: Nunito(标题) + Inter(正文)

## 快速开始

### 🚀 方式一：直接预览（无需安装，推荐）

直接用浏览器打开项目根目录下的 **`index-preview.html`** 文件即可立即体验！

**特点：**
- ✅ 无需安装 Node.js
- ✅ 无需运行任何命令
- ✅ 包含所有核心功能
- ✅ 数据自动保存到浏览器

### 🛠️ 方式二：使用开发环境（需要 Node.js）

#### 1. 安装 Node.js

如果尚未安装 Node.js，请先安装：
- 访问 https://nodejs.org/
- 选择 LTS 版本（推荐 18.x 或 20.x）
- 安装时确保勾选 "Add to PATH" 选项

#### 2. 安装依赖

```bash
npm install
```

#### 3. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173 查看应用

#### 4. Windows 用户快速启动

双击运行项目根目录下的 **`start.bat`** 文件即可自动完成安装和启动。

#### 5. 构建生产版本

```bash
npm run build
```

#### 6. 预览生产版本

```bash
npm run preview
```

### ⚠️ 如果 `start.bat` 无法运行

**可能原因和解决方法：**

1. **Node.js 未安装**
   - 访问 https://nodejs.org/ 下载并安装 LTS 版本
   - 安装完成后重新启动命令提示符

2. **Node.js 未添加到 PATH**
   - 卸载并重新安装 Node.js，确保勾选 "Add to PATH"
   - 或手动将 Node.js 安装目录添加到系统环境变量 PATH

3. **脚本窗口一闪而过**
   - 打开命令提示符（CMD）
   - 切换到项目目录：`cd d:\1sqh_files\AI_code_practice0528-0607\0601test`
   - 手动执行：`start.bat`
   - 这样可以看到详细的错误信息

## 使用说明

### 创建任务
1. 点击右上角的"添加任务"按钮
2. 填写任务标题(必填)
3. 选择优先级
4. 设置截止日期和时间(可选)
5. 启用提醒并选择铃声(可选)
6. 点击"创建任务"

### 管理任务
- **切换状态**: 点击任务右上角的圆形图标
- **编辑任务**: 悬停任务卡片,点击编辑图标
- **删除任务**: 悬停任务卡片,点击删除图标

### 筛选任务
- 使用左侧边栏的状态筛选器
- 按日期分类查看任务
- 切换排序方式

## 浏览器兼容性

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 注意事项

1. **提醒功能**: 需要浏览器授权通知权限才能使用浏览器通知
2. **数据存储**: 所有数据保存在浏览器本地存储中,清除浏览器数据会导致任务丢失
3. **铃声播放**: 部分浏览器可能需要用户交互后才能播放音频
